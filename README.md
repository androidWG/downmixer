Download Spotify songs easily using YouTube Music. Can be an alternative or replacement to [spotDL](https://github.com/spotDL/spotify-downloader), however, it is **only a Python library, *not* a CLI tool**.

## This project is currently in alpha version.
Basic functionality mostly works, but everything else doesn't. Also, no documentation.
